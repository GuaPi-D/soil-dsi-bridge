🌐 SOIL ↔ D‑SI 桥接本体 / Bridge Ontology

📘 简体中文说明

此本体桥接了 SOIL 与 D‑SI (DCC) 本体之间的核心语义要素，以支持可查找、可获取、可互操作和可复用（FAIR）的测量数据交换。

版本： 1.0

创建时间： 2025-05-20

作者： Zeyuan D., 王康

📄 English Description

This ontology bridges core semantic elements between the SOIL and D‑SI (DCC) ontologies to enable FAIR (Findable, Accessible, Interoperable, Reusable) metrology data exchange.

Version: 1.0

Created: 2025-05-20

Authors: Zeyuan D., Kang Wang
